HTML Cheat Sheet

  • <h1>...</h1> - Koptekst
  • <p>...</p> - Paragraaf
  • <a href="url">...</a> - Link
  • <img src="url" alt="tekst"> - Afbeelding
  • <ul><li>...</li></ul> - Ongeordende lijst
  • <ol><li>...</li></ol> - Geordende lijst
  • <input type="text"> - Tekstinvoer
  • <form>...</form> - Formulier
  • <table>...</table> - Tabel
  • <div>...</div> - Container
  • <span>...</span> - Inline container
  • <meta charset="UTF-8"> - Karakterset

CSS Cheat Sheet

  • color: tekstkleur
  • background-color: achtergrondkleur
  • margin: buitenruimte
  • padding: binnenruimte
  • border: rand
  • display: flex; - Flexbox layout
  • display: grid; - Grid layout
  • font-size: lettergrootte
  • box-shadow: schaduw
  • border-radius: afgeronde hoeken
  • transition: animatie-overgang
  • :hover - Hover effect
  • position: absolute; - Absolute positionering
  • z-index: stapelvolgorde
  • overflow: hidden; - Overloop verbergen

JavaScript Cheat Sheet

  • let x = 5; - Variabele declareren
  • const PI = 3.14; - Constante
  • function naam() { ... } - Functie declareren
  • console.log('Hallo'); - Print naar console
  • document.getElementById('id') - Element selecteren
  • if (voorwaarde) { ... } - Voorwaardelijke logica
  • for (let i = 0; i < 5; i++) { ... } - For-loop
  • let arr = [1,2,3]; arr.forEach(fn); - Array iteratie
  • addEventListener('click', fn) - Event listener
  • fetch(url).then(...) - Data ophalen
  • try { ... } catch(e) { ... } - Foutafhandeling
  • setTimeout(fn, 1000); - Timer
  • JSON.parse(json) - JSON omzetten naar object

SQL Cheat Sheet

  • SELECT * FROM tabel; - Selecteer alles
  • INSERT INTO tabel (kolom) VALUES (waarde); - Voeg toe
  • UPDATE tabel SET kolom=waarde WHERE ...; - Update
  • DELETE FROM tabel WHERE ...; - Verwijder
  • CREATE TABLE naam (...); - Maak tabel
  • ALTER TABLE naam ADD kolom datatype; - Wijzig tabel
  • DROP TABLE naam; - Verwijder tabel

TypeScript Cheat Sheet

  • let x: number = 5; - Variabele met type
  • function greet(name: string): void { ... } - Functie met types
  • interface Persoon { naam: string; leeftijd: number; } - Interface
  • type ID = number | string; - Type alias
  • class Auto { ... } - Klasse
  • const arr: string[] = []; - Array met type
  • enum Kleur { Rood, Groen, Blauw } - Enum
  • let onbekend: unknown; - Onbekend type
  • as - Type casting (x as number)
