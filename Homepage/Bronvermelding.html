Uitleg: HTML Search Bar maken

Deze zoekbalk wordt gebruikt op de Cheat Sheets pagina van deze site. Je kunt hiermee live filteren op alle programmeertalen:

HTML

<input type="text" id="cheatSearch" placeholder="Zoek in alle cheat sheets..." onkeyup="filterCheatSheets()">

JavaScript

function filterCheatSheets() {
    const input = document.getElementById('cheatSearch').value.toLowerCase();
    document.querySelectorAll('.cheat-card').forEach(card => {
        const text = card.innerText.toLowerCase();
        card.style.display = text.includes(input) ? '' : 'none';
    });
}

Meer uitleg over zoekbalken vind je op W3Schools: Filter List.

Uitleg: CSS Button Hover Effect

Dit is het hover-effect van de blauwe knoppen op deze site:

HTML

<a href="#" class="cta-btn">Klik hier</a>

CSS

.cta-btn {
    background-color: #3498db;
    color: white;
    padding: 12px 24px;
    border-radius: 6px;
    text-decoration: none;
    display: inline-block;
    transition: background 0.3s, transform 0.3s, box-shadow 0.3s;
}

.cta-btn:hover {
    background-color: #2980b9;
    transform: translateY(-3px) scale(1.04);
    box-shadow: 0 4px 12px rgba(52,152,219,0.25);
}

Meer uitleg over CSS hover effecten vind je op MDN Web Docs: :hover CSS pseudo-class.
